# Prédiction des prix de l’or avec ARIMA

## Description
Ce projet a pour objectif de prédire les prix de l’or à partir de données historiques en utilisant le modèle ARIMA.  
Une API backend a été développée avec FastAPI pour exposer le modèle, et une interface frontend avec React.js permet de visualiser les prédictions.

## Technologies utilisées
- Python
- FastAPI
- React.js
- Pandas
- NumPy
- Matplotlib 
- Statsmodels (ARIMA)

## Architecture du projet
- Backend (FastAPI) : entraînement du modèle et API de prédiction
- Frontend (React.js) : interface utilisateur pour afficher les résultats

## Méthodologie
- Collecte et nettoyage des données historiques
- Analyse des séries temporelles
- Entraînement du modèle ARIMA
- Évaluation des performances
- Déploiement du modèle via une API

## Fonctionnalités
- Prédiction des prix futurs de l’or
- Visualisation des données historiques
- Interaction via une interface web

## Résultats
Le modèle ARIMA permet de capturer les tendances des séries temporelles et de générer des prédictions cohérentes.
